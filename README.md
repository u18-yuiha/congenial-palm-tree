# このプログラムができること
入力された動画から一定の音量以下の部分を検出して、その部分をカットした動画を出力します。

＊動画をカットする際に「プツッ」としたノイズが乗る場合があります。
気になる場合は動画編集ソフトなどでBGMを追加してあげてください。
